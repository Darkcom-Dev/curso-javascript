El "curso" muy entre comillas será presentado en un repositorio de GitHub, donde se encuentra alojado este HTML que está visualizando desde su navegador. Mas bien este es un repositorio de apuntes personales que espero que a otras personas les pueda ayudar. Todo el contenido aquí presentado, tanto los scripts, como las hojas de estilo, como los scripts de JS tendran comentarios explicativos, asi que no se quede con lo superficial. Pretendia presentar este curso en cuadernos de Jupyter, pero me pareció que no era adecuado, tambien pensé en crear una página web dinamica como si fuera una web de cursos, pero tengo problemas con el despliegue, con los costos, asi que voy a usar todo el poder de las webs estáticas y GitHub para presentar el contenido.
